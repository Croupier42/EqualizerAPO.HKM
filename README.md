Что-либо не нравится в работе скрипта или есть идеи как его улучшить - пишите, возможно смогу реализовать
## Возможности
- Проверка на наличие Equalizer APO во время первоначальной настройки
- Автозагрузка скрипта при запуске системы\
	при включении создается, а при выключении удаляется ярлык скрипта в\
	`"%AppData%\Microsoft\Windows\Start Menu\Programs\Startup"`
- Переключение между пятью профилями или устройствами по горячим клавишам\
	Volume_Mute и Media_Stop `у меня Fn+F9 и Fn+F10`
- При нажатии мультимедиа кнопок громкости Volume_Down и Volume_Up `у меня Fn+F11 и Fn+F12`\
	Фиксируется системная громкость на 100%\
	Регулируется громкость через Equalizer APO
- Включение VST плагина при привышении 0дБ\
	Подразумевается плагин [лимитера](https://loudmax.blogspot.com/) или [авторегулировки громкости](https://www.hornetplugins.com/plugins/hornet-vu-meter-mk4/), ссылки приведены как пример
- Запуск `"C:\Program Files\EqualizerAPO\Editor.exe"` по горячей клавише AltGr+Volume_Up (у меня Fn+AltGr+F12)\
	также запускается `mmsys.cpl` если выбран режим переключения между устройствами
